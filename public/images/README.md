# Bilder for Å i Lofoten nettsiden

Legg bildene dine i denne mappen med følgende navn:

## Påkrevde bilder:

- `hero.jpg` - Hovedbilde for landingssiden (bredt format, anbefalt 1920x1080 eller større)
- `fysisk-infrastruktur.jpg` - Bilde for fysisk infrastruktur tiltakspakke
- `vertskap-formidling.jpg` - Bilde for vertskap og formidling tiltakspakke
- `organisering-kompetanse.jpg` - Bilde for organisering og kompetanse tiltakspakke
- `bakgrunn.jpg` - Bilde for bakgrunnsiden
- `toaletter-hygiene.jpg` - Bilde for toaletter og hygiene tiltak
- `parkering-trafikk.jpg` - Bilde for parkering og trafikk tiltak
- `informasjonssystemer.jpg` - Bilde for informasjonssystemer tiltak
- `naturvern-baerekraft.jpg` - Bilde for naturvern og bærekraft tiltak
- `placeholder.jpg` - Fallback-bilde hvis noen bilder mangler (valgfritt)

## Støttede formater:
- JPG/JPEG
- PNG
- WebP (anbefalt for bedre ytelse)

## Tips:
- Optimaliser bildene før du legger dem inn (bruk f.eks. TinyPNG eller ImageOptim)
- Hero-bildet bør være høy kvalitet og bredt format
- Tiltakspakkene kan være kvadratiske eller vertikale (anbefalt 800x600 eller større)

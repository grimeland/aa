# Å i Lofoten - Besøksforvaltning

Website for besøksforvaltning på Å i Lofoten. Prosjektet presenterer prinsipper og tiltak for bærekraftig turisme.

## Teknologi

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Komme i gang

1. Installer avhengigheter:
```bash
npm install
```

2. Kjør utviklingsserver:
```bash
npm run dev
```

3. Åpne [http://localhost:3000](http://localhost:3000) i nettleseren.

## Struktur

- `/` - Landingsside med hero, prinsipper og tiltakspakker
- `/tiltak` - Oversikt over alle tiltakspakker
- `/tiltak/fysisk-infrastruktur` - Tiltak for fysisk infrastruktur
- `/tiltak/vertskap-formidling` - Tiltak for vertskap og formidling
- `/tiltak/organisering-kompetanse` - Tiltak for organisering og kompetanse
- `/bakgrunn` - Om prosjektet
- `/folkene` - Om prosjektteamet
- `/kontakt` - Kontaktinformasjon

## CMS

Strukturen er satt opp slik at det enkelt kan kobles til et CMS senere. Innholdet kan enkelt flyttes til CMS-baserte løsninger.
